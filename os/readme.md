## 运行环境
本项目中，之前的代码，除ex17.asm外，都是在dosbox模拟环境下运行测试的。本次课程设计，和之前不同。之前的代码都只是dos下的一个程序而已，而本次设计的代码，是一个简单的操作系统。编写风格及测试环境都有所不同。
